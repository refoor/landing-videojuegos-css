# Landing Page de Videojuegos

Este fue uno de mis primeros proyectos web, desarrollado para practicar HTML y CSS.

## Descripción

Página web con temática de videojuegos, creada como práctica inicial para aprender estructura de páginas, diseño visual, uso de imágenes, menús de navegación, botones y secciones.

## Tecnologías utilizadas

- HTML
- CSS

## Objetivo del proyecto

El objetivo principal fue practicar el diseño de una página web desde cero, aplicando estilos con CSS y organizando el contenido mediante HTML.

## Lo que aprendí

- Crear una página web con HTML.
- Aplicar estilos con CSS.
- Usar imágenes, fondos y botones.
- Crear un menú de navegación.
- Organizar secciones dentro de una página.
- Mejorar la presentación visual de un sitio web.
